## laf-cli

### Install

```bash
npm i laf-cli -g
```

## Usage

### 1. login your account

```bash
  laf login -u maslow -p password -r https://console.lafyun.com
```

### 2. show your applications list

```bash
  laf list
```

### 3. init in your local project

```bash
  laf init -s YOUR_APPID
```

## Upgrade laf-cli

```sh
  npm i laf-cli@latest -g
```
